# 🍅 POMO

ポモドーロテクニックを統合したTODO管理アプリです。

## ✨ 特徴

- 📝 **TODOリスト管理**: タスクの追加、完了、削除が簡単
- ⏱️ **ポモドーロタイマー**: 各タスクに25分の作業タイマーと5分の休憩タイマー
- 🎯 **ポモドーロ見積もり**: タスクごとに必要なポモドーロ数を設定可能
- 📊 **進捗管理**: 完了/総タスク数と完了率をリアルタイム表示
- 🎉 **完了演出**: 全タスク完了時に華やかなお祝い演出
- 📅 **ICSファイル対応**: カレンダーファイルから今日のTODOを自動読み込み
- 💾 **データ永続化**: ブラウザのlocalStorageに自動保存
- 🎨 **直感的なUI**: モダンでレスポンシブなデザイン

## 🎯 主な機能

### ポモドーロタイマー
- ⏯️ 開始/一時停止/リセット
- 🔵 作業フェーズ（25分）と🟢 休憩フェーズ（5分）の自動切替
- 🔝 タイマー開始時に該当タスクがリストの最上位に移動
- 🚫 同時に実行できるタイマーは1つのみ（シングルタイマーモード）
- ✅ タスク完了時にタイマー自動停止

### タスク管理
- ➕➖ ポモドーロ数の増減ボタン
- 📊 各タスクの進捗バー表示
- 🎖️ 完了タスクに「DONE!」バッジ表示
- 👻 完了タスクは半透明表示＋コントロール無効化
- 📈 全体の進捗サマリー（分数表示＋パーセント）

### ICSファイル読み込み
- 📁 カレンダー形式（.ics）のファイルをサポート
- 📅 今日が期限のタスクのみを自動抽出
- ✅ 完了済みタスクは自動除外
- ⏰ 時間情報（PT1H等）からポモドーロ数を自動推定

### お祝い演出
- 🎊 全タスク完了時に「ALL DONE!」メッセージ
- 🎉 カラフルな紙吹雪アニメーション
- ✨ 背景が虹色に輝くエフェクト
- ⏱️ 5秒後に自動で演出終了

## 🚀 使い方

### 開発環境のセットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build

# ビルド結果のプレビュー
npm run preview
```

### アプリの使い方

1. **タスクの追加**
   - 入力欄にタスク名を入力して「追加」ボタンをクリック
   - 新しいタスクは一番上に追加されます

2. **ポモドーロ数の設定**
   - 各タスクの「-」「+」ボタンで見積もりポモドーロ数を調整

3. **タイマーの使用**
   - 「開始」ボタンでタイマースタート（タスクが最上位に移動）
   - 「一時停止」で中断、「リセット」で初期状態に戻る
   - 作業25分→休憩5分が自動で切り替わります

4. **ICSファイルの読み込み**
   - 「📅 ICSファイルから今日のTODOを読み込む」ボタンをクリック
   - .icsファイルを選択すると、今日のタスクが自動追加されます

5. **タスクの完了**
   - チェックボックスをクリックでタスク完了
   - 全タスク完了でお祝い演出が発動！

## 🛠️ 技術スタック

- **React 18.3.1**: UIライブラリ
- **Vite 6.0.7**: ビルドツール
- **CSS3**: スタイリング（アニメーション、グラデーション）
- **localStorage**: データ永続化

## 📦 データ構造

```javascript
{
  id: Number,              // ユニークID
  text: String,            // タスク名
  completed: Boolean,      // 完了状態
  estimatedPomodoros: Number,   // 見積もりポモドーロ数
  completedPomodoros: Number,   // 完了したポモドーロ数
  remainingTime: Number,   // 残り時間（秒）
  currentPhase: 'work' | 'break',  // 現在のフェーズ
  isRunning: Boolean       // タイマー実行中か
}
```
